# Underground Street Racing RPG

A Java desktop application built using JavaFX that implements a role-playing game (RPG) centered around underground street racing. Decoupled using MVC and built with Gradle.

## Technical Specifications
- **Language**: Java 17 or newer (tested on Java 23)
- **GUI**: JavaFX
- **Persistence**: JSON files utilizing Jackson
- **Build Tool**: Gradle

## Execution Instructions
To download, compile, and run the project, use the following two commands:

1. **Build the project**:
   ```bash
   ./gradlew build
   ```
   *(On Windows PowerShell, use `.\gradlew.bat build`)*

2. **Run the application**:
   ```bash
   ./gradlew run
   ```
   *(On Windows PowerShell, use `.\gradlew.bat run`)*
